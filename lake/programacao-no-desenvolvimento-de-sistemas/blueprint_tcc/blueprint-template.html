Como usar este documento
  • Preencha cada uma das 8 seções no seu ritmo. Tudo salva sozinho neste navegador, neste computador.

  • Use 💾 Salvar arquivo para baixar um .json com tudo — é seu backup e o que você entrega ao professor. Use 📂 Abrir para voltar de onde parou em outro computador.
  • Quando estiver pronto, use 🖨️ Exportar PDF (a barra e estas dicas somem no PDF).
