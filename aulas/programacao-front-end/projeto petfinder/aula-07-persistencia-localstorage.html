Aula 07 · Projeto PetFinder

Dados que não somem, sem servidor: localStorage

O sistema da Aula 06 perde tudo no F5 — os pets vivem só na memória. Hoje resolvemos isso usando o localStorage, um pequeno espaço de armazenamento que já vem dentro de todo navegador. Sem instalar nada, sem Node, sem servidor: o projeto continua sendo só os três arquivos do front-end, e os pets passam a sobreviver a F5, ao fechar o navegador e a desligar o computador.

Seção 01

Introdução

O sistema da Aula 06 funciona, mas tem um problema grave: ao apertar F5 ou fechar a janela, todos os pets desaparecem. É esperado — eles estão só na memória do navegador. Existem várias formas de resolver isso; a mais simples, e que não exige instalar absolutamente nada, é o localStorage. Ao terminar a aula, você fecha o navegador, abre de novo, e os pets continuam lá. Tudo isso continua rodando ao abrir o index.html com um duplo clique.

Seção 02

O que vamos construir

✓

Entrega da aula

Duas funções novas no app.js — salvarPets e carregarPets — e pequenos ajustes nas funções de cadastrar, atualizar e excluir para que cada alteração seja gravada no localStorage. No fim, os pets sobrevivem a F5, ao fechamento do navegador e à reinicialização do computador. O projeto continua com apenas três arquivos no frontend: nenhum back-end, nenhum servidor.

i

Antes de começar

O projeto da Aula 06 precisa estar com o CRUD completo funcionando na memória (cadastrar, listar, editar, excluir, buscar e filtrar). Mantenha o DevTools aberto — vamos usar a aba Application → Local Storage para ver os dados salvos.

Seção 03

O que é o localStorage

i

Um cofrinho dentro do navegador

O localStorage guarda pares de chave e valor em texto, e os mantém mesmo depois de fechar o navegador. É como um cofrinho que vive na máquina, ligado àquele site (ou àquele arquivo).

Ele só sabe guardar texto. Como o nosso pets é uma lista de objetos, usamos JSON.stringify(pets) para transformar a lista em texto na hora de salvar, e JSON.parse(...) para transformar o texto de volta em lista na hora de ler. Três comandos resumem tudo: localStorage.setItem(chave, texto) grava, localStorage.getItem(chave) lê e localStorage.removeItem(chave) apaga.

!

Importante: é por máquina, não compartilhado

O localStorage fica no navegador daquele computador. O que o aluno salva no PC 12 não aparece no PC 13 — cada máquina tem seu próprio cofrinho. Para um banco compartilhado pela turma seria preciso um servidor, o que foge do escopo desta aula.

Seção 04

Passo a passo

Boa notícia: o HTML e o CSS não mudam nada. Todo o trabalho é no app.js, e é pouco.

1. Adicionar a chave e as funções de persistência

No topo do app.js, junto das outras constantes, defina o nome da "gaveta" onde os dados ficam. Em seguida, no corpo do arquivo, adicione as duas funções:

javascript 
const CHAVE = 'petfinder-pets';

function salvarPets() {
  localStorage.setItem(CHAVE, JSON.stringify(pets));
}

function carregarPets() {
  pets = JSON.parse(localStorage.getItem(CHAVE)) || [];
  mostrarPetsNaTela();
}

2. Salvar a cada cadastro e edição

Faça as funções cadastrarPet e atualizarPet chamarem salvarPets() sempre que mudarem a lista:

javascript 
function cadastrarPet(pet) {
  pets.push({ id: Date.now(), ...pet });
  salvarPets();
}

function atualizarPet(id, dados) {
  pets = pets.map(function (p) {
    return p.id === id ? { id, ...dados } : p;
  });
  salvarPets();
}

3. Salvar também ao excluir

Na excluirPet, grave a lista logo depois de remover o pet:

javascript 
function excluirPet(id) {
  const pet = pets.find(function (p) { return p.id === id; });
  if (!pet) return;
  if (!confirm(`Deseja excluir o cadastro de ${pet.nome}?`)) return;
  pets = pets.filter(function (p) { return p.id !== id; });
  salvarPets();
  mostrarPetsNaTela();
  mostrarMensagem('Pet excluído.', 'sucesso');
}

4. Carregar os pets ao abrir a página

No final do app.js, troque qualquer chamada antiga de carregamento por esta — ela lê o que estava salvo e desenha na tela:

javascript 
carregarPets();
Seção 05

Checkpoint e desafio

★

Sua página passa se…

Você cadastra dois ou três pets, aperta F5 e eles continuam lá; fecha o navegador, abre de novo, e os pets seguem na tela; editar e excluir também persistem após recarregar; e, no DevTools → Application → Local Storage, aparece a chave petfinder-pets com o texto JSON dos pets.

★

Desafio (até 10 min)

Crie um botão Limpar tudo no cabeçalho que, após um confirm, apague todos os pets de uma vez. Dica: use localStorage.removeItem('petfinder-pets'), zere o array com pets = [] e chame mostrarPetsNaTela().

Seção 06

Erros comuns

!

Os pets somem ao recarregar, mesmo com o código novo

Sintoma: cadastra, dá F5 e a lista volta vazia. Causa: faltou chamar salvarPets() dentro de quem altera a lista, ou a chave usada para salvar é diferente da usada para ler. Solução: confira se cadastrarPet, atualizarPet e excluirPet chamam salvarPets(), e se a mesma constante CHAVE é usada no setItem e no getItem.

!

"Unexpected token" ou tela em branco ao abrir

Sintoma: a página nem carrega a lista e o console mostra erro de JSON.parse. Causa: o texto salvo no localStorage está corrompido (alteração manual ou versão antiga). Solução: proteja a leitura com || [] (como no código), e, se preciso, limpe a chave rodando no console: localStorage.removeItem('petfinder-pets'). Para inspecionar o que está guardado:

javascript 
localStorage.getItem('petfinder-pets');
Seção 07

Código completo

O app.js inteiro na versão com localStorage — é o CRUD da Aula 06 com a persistência somada. Cole por cima do arquivo atual; o HTML e o CSS da Aula 06 continuam valendo sem mudança:

javascript 
// ===== Referências de elementos =====
const formulario = document.querySelector('#pet-form');
const listaPets = document.querySelector('#lista-pets');
const mensagem = document.querySelector('#mensagem');
const contadorPets = document.querySelector('#contador-pets');
const campoBusca = document.querySelector('#busca');
const filtroEspecie = document.querySelector('#filtro-especie');
const botaoSalvar = document.querySelector('#botao-salvar');
const botaoCancelar = document.querySelector('#botao-cancelar');
const campoId = document.querySelector('#pet-id');

const CHAVE = 'petfinder-pets';

let pets = [];
let editandoId = null;

// ===== Eventos =====
campoBusca.addEventListener('input', mostrarPetsNaTela);
filtroEspecie.addEventListener('change', mostrarPetsNaTela);
botaoCancelar.addEventListener('click', limparFormulario);

formulario.addEventListener('submit', function (evento) {
  evento.preventDefault();
  limparErrosVisuais();

  const pet = montarPetDoFormulario();
  const erro = validarPet(pet);
  if (erro) { mostrarMensagem(erro, 'erro'); return; }

  if (editandoId) {
    atualizarPet(editandoId, pet);
    mostrarMensagem('Pet atualizado!', 'sucesso');
  } else {
    cadastrarPet(pet);
    mostrarMensagem('Pet cadastrado!', 'sucesso');
  }

  limparFormulario();
  mostrarPetsNaTela();
});

// ===== Persistência no localStorage =====
function salvarPets() {
  localStorage.setItem(CHAVE, JSON.stringify(pets));
}

function carregarPets() {
  pets = JSON.parse(localStorage.getItem(CHAVE)) || [];
  mostrarPetsNaTela();
}

// ===== Criar e atualizar (sempre salvando) =====
function cadastrarPet(pet) {
  pets.push({ id: Date.now(), ...pet });
  salvarPets();
}

function atualizarPet(id, dados) {
  pets = pets.map(function (p) {
    return p.id === id ? { id, ...dados } : p;
  });
  salvarPets();
}

// ===== Coleta e validação =====
function montarPetDoFormulario() {
  return {
    nome: document.querySelector('#nome').value.trim(),
    especie: document.querySelector('#especie').value,
    idade: document.querySelector('#idade').value,
    cidade: document.querySelector('#cidade').value.trim(),
    telefone: document.querySelector('#telefone').value.trim(),
    foto: document.querySelector('#foto').value.trim(),
    descricao: document.querySelector('#descricao').value.trim(),
    status: document.querySelector('#status').value
  };
}

function validarPet(pet) {
  if (pet.nome.length < 2) {
    document.querySelector('#nome').classList.add('campo-invalido');
    return 'O nome precisa ter pelo menos 2 letras.';
  }
  if (!pet.especie) {
    document.querySelector('#especie').classList.add('campo-invalido');
    return 'Escolha a espécie do pet.';
  }
  if (pet.idade && Number(pet.idade) < 0) {
    document.querySelector('#idade').classList.add('campo-invalido');
    return 'A idade não pode ser negativa.';
  }
  if (pet.telefone && pet.telefone.length < 8) {
    document.querySelector('#telefone').classList.add('campo-invalido');
    return 'Digite um telefone com pelo menos 8 números.';
  }
  return null;
}

// ===== Mensagens e limpeza =====
function mostrarMensagem(texto, tipo) {
  mensagem.textContent = texto;
  mensagem.className = 'mensagem ' + tipo;
}

function limparErrosVisuais() {
  const campos = document.querySelectorAll('input, select, textarea');
  for (const campo of campos) campo.classList.remove('campo-invalido');
}

function limparFormulario() {
  formulario.reset();
  campoId.value = '';
  editandoId = null;
  botaoSalvar.textContent = 'Cadastrar pet';
  botaoCancelar.classList.add('escondido');
  limparErrosVisuais();
}

// ===== Renderização =====
function mostrarPetsNaTela() {
  listaPets.innerHTML = '';
  const termo = campoBusca.value.toLowerCase();
  const especie = filtroEspecie.value;

  const filtrados = pets.filter(function (p) {
    const nomeOk = p.nome.toLowerCase().includes(termo);
    const especieOk = !especie || p.especie === especie;
    return nomeOk && especieOk;
  });

  atualizarContador(filtrados.length);
  for (const pet of filtrados) listaPets.appendChild(criarCardPet(pet));
}

function criarCardPet(pet) {
  const card = document.createElement('article');
  card.classList.add('card-pet');
  const imagem = pet.foto || 'https://placehold.co/600x400?text=Sem+foto';
  const classeStatus = pet.status === 'Adotado' ? 'status adotado' : 'status';

  card.innerHTML = `
    <img src="${imagem}" alt="${pet.nome}">
    <div class="card-pet-conteudo">
      <h3>${pet.nome}</h3>
      <p>Espécie: ${pet.especie}</p>
      <p>Cidade: ${pet.cidade || 'Sem cidade'}</p>
      <span class="${classeStatus}">${pet.status}</span>
      <div class="acoes-card">
        <button class="botao-editar" type="button">Editar</button>
        <button class="botao-excluir" type="button">Excluir</button>
      </div>
    </div>
  `;

  card.querySelector('.botao-editar').addEventListener('click', () => carregarPetParaEdicao(pet.id));
  card.querySelector('.botao-excluir').addEventListener('click', () => excluirPet(pet.id));
  return card;
}

function atualizarContador(total) {
  if (total === 0) contadorPets.textContent = 'Nenhum pet encontrado.';
  else if (total === 1) contadorPets.textContent = '1 pet encontrado.';
  else contadorPets.textContent = `${total} pets encontrados.`;
}

// ===== Edição e exclusão =====
function carregarPetParaEdicao(id) {
  const pet = pets.find(function (p) { return p.id === id; });
  if (!pet) return;
  editandoId = id;
  campoId.value = pet.id;
  document.querySelector('#nome').value = pet.nome;
  document.querySelector('#especie').value = pet.especie;
  document.querySelector('#idade').value = pet.idade;
  document.querySelector('#cidade').value = pet.cidade;
  document.querySelector('#telefone').value = pet.telefone;
  document.querySelector('#foto').value = pet.foto;
  document.querySelector('#descricao').value = pet.descricao;
  document.querySelector('#status').value = pet.status;
  botaoSalvar.textContent = 'Salvar alterações';
  botaoCancelar.classList.remove('escondido');
  window.scrollTo({ top: 0, behavior: 'smooth' });
}

function excluirPet(id) {
  const pet = pets.find(function (p) { return p.id === id; });
  if (!pet) return;
  if (!confirm(`Deseja excluir o cadastro de ${pet.nome}?`)) return;
  pets = pets.filter(function (p) { return p.id !== id; });
  salvarPets();
  mostrarPetsNaTela();
  mostrarMensagem('Pet excluído.', 'sucesso');
}

// ===== Início =====
carregarPets();

O que levamos desta aula

  • Você entendeu a diferença entre dados na memória (somem no F5) e dados persistidos (sobrevivem).
  • Conheceu o localStorage e seus três comandos: setItem, getItem e removeItem.
  • Usou JSON.stringify para guardar uma lista como texto e JSON.parse para lê-la de volta.
  • Centralizou a gravação numa função salvarPets(), chamada sempre que a lista muda — um padrão limpo e fácil de manter.
