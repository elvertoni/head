## 1. O modelo pede, o sistema executa

- **secao:** Desenvolvimento › A IA não tem mãos — ela pede (callout `:::atencao`)
- **objetivo:** Fixar a ideia-chave: o LLM não executa, ele faz um pedido estruturado e o sistema executa. Concretiza com a analogia do chef que grita o pedido.
- **alt:** À esquerda, o LLM (sem mãos) emitindo um cartão de pedido "usar calculadora: 1234x5678"; à direita, o "sistema" com uma calculadora executando e devolvendo o resultado ao LLM.
- **prompt:** Left: an LLM character without hands emitting a structured request card "usar calculadora: 1234 x 5678". Right: a "sistema" box with a real calculator computing and sending the result back via an arrow. Flat vector, conveys "model asks, system does", minimal text. 16:9.
